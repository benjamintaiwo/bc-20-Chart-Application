# BC-21-Chart-Application
## Chart Application Powered by HTML5 and JAVASCRIPT
This app allows you to plot different charts such as line graph, bar chart, pie chart etc

# FEATURES
* Allow users to input a set of data
* Allow Users to enter the title of the chart
* Allow users to select the type of chart
* Also users are allowed to plot data and clear the data set
* Allows Users print, download the chart (available in next release)
